Rick and Morty Website Este projeto foi desenvolvido como parte de um mini-curso introdutório, focado em ensinar os fundamentos de HTML, CSS e JavaScript. Durante as aulas, os tutores explicaram passo a passo como criar um site simples inspirado na temática de "Rick and Morty", utilizando conceitos básicos de desenvolvimento web. O principal objetivo foi aprender e aplicar: Estruturação de páginas web com HTML. Estilização e design responsivo com CSS. Uso básico de JavaScript para interatividade e manipulação de conteúdo. Observação Este site foi desenvolvido acompanhando as instruções dos tutores durante o curso, ou seja, ele não é 100% autoral, mas sim um exercício guiado. O foco foi entender as tecnologias e boas práticas no desenvolvimento web. Tecnologias Utilizadas: HTML5 CSS3 JavaScript Embora o projeto tenha sido uma introdução, pretendo expandi-lo no futuro adicionando: Integração com uma API para carregar informações dinâmicas. Mais interatividade com JavaScript avançado. Melhorias no design e experiência do usuário.
